A minimalistic and responsive web-based calculator application developed using HTML, CSS, and JavaScript. The application features a modern dark-themed user interface suitable for both desktop and mobile devices.

Features
Live Preview: Displays real-time calculation results while typing.
Dark Interface: Designed with a gradient dark theme for visual comfort.
Responsive Layout: Automatically adjusts layout for mobile and desktop screens.
Standard Operations: Supports addition, subtraction, multiplication, and division.
Input Control: Includes clear all (AC) and backspace (Del) functionality.

Technologies Used
HTML5: Application structure.
CSS3: Styling, grid layout, and responsive design.
JavaScript: calculation logic and DOM manipulation.

Usage Instructions
Save the source code as index.html.
Open the file in any modern web browser (Chrome, Firefox, Edge, etc.).
The application is ready for immediate use.
